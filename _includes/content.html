프로젝트 상세

    포털 사이트, SNS 검색을 통해 커피숍을 찾는 사용자가 대다수입니다.
    하지만 광고성 글과 무분별한 정보들이 많이 있어 정보를 찾기가 힘들어져서
    Spring Frame Work, Spring Boot를 활용하여 커피숍 정보 검색 웹 서비스 사이트를 만들었습니다.
    oracleDB는 Spring JDBC를 통해 연동하였습니다.
    HTML,CSS,JS,Bootstrap를 기반으로 제작되었고, AJAX와 REST API를 활용하였습니다.

역할

    대부분의 컨트롤러(Java)와 검색바의 디자인 및 JavaScript 제작, Thymeleaf를 이용한 화면을 구현했습니다.
    첨부파일과 관련된 DB를 설계하였고, 커피숍 사진정보에 대한 CRUD 기능을 구현했습니다.

소요기술

테이블 구성(ERD)

SW 아키텍처
